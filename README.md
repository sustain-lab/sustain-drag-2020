# sustain-drag-2020

## Getting started

### Get the code

```
git clone https://github.com/sustain-lab/sustain-drag-2020
cd sustain-drag-2020
```

### Set up your Python environment

```
python3 -m venv venv
source venv/bin/activate
pip install -U pip
pip install -U -r requirements.txt
```
